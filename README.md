# RecMeet 🚀
### Record & Manage Your Classes Effortlessly

RecMeet is an all-in-one EdTech SaaS platform designed specifically for teachers and coaching institutes. We simplify the entire process of class recording, student management, and parent communication.

---

## ✨ Core Features (The 5 Pillars)
1. **Teacher Operations**: Sleek dashboard for lecture and marks management.
2. **Student Learning Portal**: Dedicated space for watching recordings and tracking performance.
3. **Parent Tracking**: Automated progress updates and fee reminders.
4. **AI Doubt Solver**: 24/7 integrated AI support to help students solve doubts instantly.
5. **Automated Alerts**: WhatsApp and call notifications for important updates.

## 🛠 Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Backend**: Node.js, Express.
- **Database**: Hybrid support for SQLite (local) and MongoDB (production).
- **Icons**: Lucide Icons.
- **Animations**: Custom CSS Keyframes & Intersection Observer API.

## 🚀 Quick Start (Local)
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   node server.js
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🌍 Deployment
This project is production-ready and can be deployed on platforms like **Render**, **Heroku**, or **Vercel**.
- **Database**: Connect to MongoDB Atlas by setting the `MONGO_URI` environment variable.
- **Hosting**: Recommended to use Render for the Node.js backend.
